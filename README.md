# Fiorano Cloud Native Integration – Associate Developer

This repository contains labs, exercises, and implementation examples completed as part of the **Fiorano Cloud Native Integration Associate Developer** learning path. The work in this repository demonstrates practical experience with cloud-native integration concepts using the Fiorano integration platform.

The projects focus on designing and implementing **event-driven architectures, microservice-based integrations, and API-driven workflows**. Through these exercises, various integration patterns are explored, including service orchestration, messaging, and API exposure.

## Key Learning Areas
- Cloud-native integration architecture
- Event-driven application design
- Microservice integration using Fiorano
- API development and management
- Service orchestration and messaging workflows
- Version control for integration projects using Git

This repository serves as both a **learning record and a practical reference** for building scalable and modular integration solutions using Fiorano technologies.
