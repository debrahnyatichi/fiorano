# Fiorano Cloud Native Integration – Associate Developer

This repository contains labs, exercises, and integration workflows developed as part of the **Fiorano Cloud Native Integration Associate Developer** learning path. The projects demonstrate practical implementation of **cloud-native integration, event-driven architecture, and microservice orchestration** using the Fiorano integration platform.

The objective of this repository is to showcase hands-on experience in designing and implementing scalable integration solutions using modern enterprise integration patterns.

---

## Architecture Overview

The solutions implemented in this repository follow a **cloud-native and event-driven integration architecture** where independent services communicate through APIs and messaging channels.

Key architectural concepts demonstrated include:

- Microservice-based integration
- Event-driven workflows
- API-based service communication
- Message routing and transformation
- Service orchestration
